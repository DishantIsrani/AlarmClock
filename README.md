# AlarmClock
An Alarm Clock GUI using python for the current time and also setting an alarm.


### About 
A python GUI to alarm clock to check the current time and also setting an alarm.


### To use the GUI run the following file 
```
main.py
```

### Requirements 
* Tkinter
* Datetime
* Pillow
